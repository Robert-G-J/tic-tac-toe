# Tic-Tac-Toe
A business logic implementation of the classic game.

# User Stories
```
As a two players with free time
So that we can enjoy ourselves
We want to play a game of TicTacToe

As players
So that we can continue playing
We want to switch turns

As player 1 
So that I can start to win 
I want to place a X in a free field

As player 2
I want to place an O in a free field 
So that I can start winning 

As players
We want to only place X & 0 in free fields
To obey the game rules

As players
So that we can win a game
We want to put three in a row

As a player
So that a game finishes
I want the game to end if all the fields are taken
```
